# PokéDex Dinámica ⚡

Proyecto Front End desarrollado para el Desafío de la Unidad 1. Es una galería dinámica que consume datos reales, estructurada con buenas prácticas y diseño responsivo.

## 🚀 Características
* **HTML5:** Estructura 100% semántica y accesible.
* **CSS3:** Diseño responsivo (CSS Grid), variables de entorno y contraste nivel AA (Modo Oscuro).
* **JavaScript (ES6+):** Consumo de la **PokéAPI** mediante `fetch` y `async/await`, con manejo de errores (`try/catch`).
* **🌟 Bonus Nivel Experto:** Incluye un buscador en vivo que filtra los resultados dinámicamente sin recargar la página.
